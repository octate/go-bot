## Getting Started

### Prerequisites

- [slack bot token](https://api.slack.com/apps)
- [golang download](https://golang.org/dl/)
- [Slacker library](https://github.com/shomali11/slacker)
  ```
   go get github.com/shomali11/slacker
  ```

### Running

```
 go run main.go
```

OR, if you using Mac/Linux

```
make
```
